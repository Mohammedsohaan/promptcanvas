# Product Blueprint

Every completed product should generate a blueprint containing the following sections.

---

## Basic Information

- Product Name
- Short Description
- Problem Statement

---

## Users

- Target Users
- User Problems
- User Goals

---

## Product Goals

- Primary Goal
- Secondary Goals

---

## Features

- Core Features
- Nice-to-Have Features

---

## MVP

The minimum version required to launch.

---

## Future Improvements

Ideas for future versions.

---

## Notes

Additional information written by the user.
