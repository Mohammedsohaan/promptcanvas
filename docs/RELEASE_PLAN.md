# Release Plan

## Version

Current Version:
v0.1.0

Target Release:
v1.0.0

---

## Version 1 Goals

Version 1 is considered complete when users can:

- Create an account
- Log in
- Create a product
- Complete the Product Wizard
- Save a product
- Edit a product
- Delete a product
- View all products

---

## Quality Checklist

Before release:

- All major bugs fixed
- Responsive on desktop
- Responsive on mobile
- Authentication working
- Database working
- Forms validated
- Loading states implemented
- Error handling implemented

---

## Deployment

Deploy using Vercel.

Verify production environment variables.

Test production before announcing the project.
