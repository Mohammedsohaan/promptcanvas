# PromptCanvas v1

## Goal

The development of PromptCanvas will produce the first functioning version of the application.

The main purpose of PromptCanvas is to allow people to think through the entire development process for a software product without having to do any of the actual coding.

---

Authentication

User Registration
User Login
User Logout

---

Dashboard

The dashboard allows users to:

- see all products
- find products
- open a product
- remove a product

---

Product

A user can create a new project or an existing one using the Project Wizard
A user can modify and/or update.

---

## Product Structure

Every product will store:

- Product Name
- Short Description
- Problem Statement
- Target Users
- Goals
- Features
- MVP
- Notes
- Created Date
- Last Updated
